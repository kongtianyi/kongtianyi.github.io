故意的意识流

发表于 2017-04-16 | 分类于 闲情偶记 | | 已有 位佳客驻足品评

过两天天气就会凉爽起来，因为今天异常的热，也许是明天。洗完澡刚换的衣服摩挲皮肤的感觉很舒服，像电视上演的婴儿的屁股。

他妈的，楼上的狗比又在敲乒乓球，虽然我已经告诫过他这会使我很不爽，真想把他的乒乓球踩个稀巴烂，把他的头拧下来让他敲。眼睛睁开我站在一条约十米宽的河边，这是个菜市场，喧嚣又杂乱，天空的颜色很正常，天地间却充斥着有节律的“咚咚”的声响，有很多人赤膊着卖鱼，同时有很多人赤膊着在河里捞鱼。这河里鱼是真多，我从没见过鱼这么多的河，我故乡的臭水沟里没有鱼，只有孑孓。这里的鱼不仅多，操他妈，能不能不敲乒乓球了，而且见人不惊，不仅不惊，而且争相往空中跃，河里捞鱼的人像在跳舞。由于最近经常做找规律的智力题，我发现了一个规律，鱼跃出来的时候口是向着天的，而且鱼越来越大，河水像传送带似的运送着它们。看着河里赤膊着的，捞的越来越开心的人，我突然有点不好的预感，我真的不想再骂人了，我X！楼上！终于跟我想的一样，一个大汉被一条跃出水面的鱼含近了嘴里，他的战斗力不足以把这条鱼扔上岸，他甚至不能从鱼嘴里出来，于是鱼就继续跃着向前顺流而下，那条大汉就一直在鱼嘴里被抛呀抛。根据我发现的规律，越来越多的人被跃出的大鱼吞进嘴里，并徒劳的想要跳出来。在我看到越来越大的鱼和鱼眼感到有些恐惧和恶心的同时，鱼的横截面积已经快要超过河面了，并感到脚下的土地正在颤动，我选择醒来。领袖发了一则说说，我还记得他前天嘲笑我说多大了还发说说。其实那天我真的应该发个说说，因为我有强烈的情感需要倾听，直到把领袖一顿乱骂后才有了缓解。王小波写过一个片段，大意是说有个人深夜去朋友家门口敲门，门开，只见朋友右手提剑，左手握着钱袋，全副武装的站在自己面前，朋友说“哦，我的朋友，你夤夜来访，必有要事，如果你遭人追杀，我马上提剑为你复仇，如果你缺钱，这里的钱你随便去使用，如果你只是清夜无聊，我的城堡里有美丽的女奴供你排遣”。那天我也是在夤夜找到了领袖，我甚至说我遭遇了艰难，可是领袖却说，“我跟老张打游戏呢，没空”，于是我就把他骂了一顿，心里好受多了。好了，我必须上楼去打人了。

姜太公钓鱼~
