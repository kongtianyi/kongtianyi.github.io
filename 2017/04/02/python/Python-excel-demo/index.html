Python读写/追加excel文件Demo

三个工具包

python操作excel的三个工具包如下，注意，只能操作.xls，不能操作.xlsx。

  • xlrd: 对excel进行读相关操作
  • xlwt: 对excel进行写相关操作
  • xlutils: 对excel读写操作的整合

这三个工具包都可以直接使用pip进行下载：

xlwt的缺陷

xlwt只能创建一个全新的excel文件,然后对这个文件进行写入内容以及保存。但是大多数情况下我们希望的是读入一个excel文件，然后进行修改或追加，这个时候就需要xlutils了。
