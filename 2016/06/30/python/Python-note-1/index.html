Python学习笔记（一）——初识Python

学习来源

本笔记的学习资源是慕课网。

Python起源

龟叔Guido van Rossum（荷兰）于1989年为了打发无聊的圣诞节而开发的编程语言。

Python的特点

  • 优雅
  • 明确
  • 简单

Python适合的领域

  • Web网站和各种网络服务
  • 系统工具和脚本
  • 作为“胶水”语言把其他语言开发的模块包装起来方便使用

Python不适合的领域

  • 贴近硬件的代码（首选C）
  • 移动开发：IOS/Android有各自的开发语言（Objc，Swift、java）
  • 游戏开发：C/C++（需高速渲染）

Python和其他语言对比

语言 类型 运行速度 代码量
C 编译为机器码 非常快 非常多
Java 编译为字节码 快 多
Python 解释执行 慢 少
  • 缺点一：速度慢
  • 缺点二：源码不能加密
