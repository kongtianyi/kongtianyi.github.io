pip简介

pip 是一个安装和管理 Python 包的工具，通过pip我们能够轻松地下载和卸载python的第三方包。

原料

  • 64位Windows10 专业版
  • python 2.7.12(没有安装python的同学请戳链接Win10下的Python环境配置)

在进入的页面右上方的搜索框输入pip，点击search按钮，如图二

在搜索列表中找到pip的最新版本，目前为pip8.1.2，点之，如图三
