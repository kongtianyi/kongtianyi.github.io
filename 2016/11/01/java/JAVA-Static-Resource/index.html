关于txt

如果仅仅是读取的话，可以考虑用上述方法打入到JAR包中，但是如果涉及到修改TXT文件的内容，就不能再打入到JAR里了，那么操作的方法就是，在源码中需要操作文件的地方直接写文件名，没有路径，然后把文件保存在JAR文件的同一目录下，就可以实现文件的操作了。当然，其它资源文件也可以用这种方法访问，但是在一些具体的情况下，还是将资源文件打入到JAR包会比较方便。
