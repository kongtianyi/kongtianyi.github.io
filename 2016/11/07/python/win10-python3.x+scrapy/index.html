win10安装python3.x+scrapy

下载安装python3

官网下载就好，https://www.python.org/downloads/release/python-352/

用installer下载比较方便，它直接把环境变量都帮你配了。

升级pip

安装好之后在cmd里执行

把pip提到最新版本

下载lxml

lxml是解析网页用的，scrapy依赖于它，它是一个第三方的库，这里推荐一个python第三方库的网站http://www.lfd.uci.edu/~gohlke/pythonlibs/，里边都是编译好的，windows下python使用者的福利。

Ctrl+F定位一下lxml

下载好之后，cmd进入下载目录，直接用pip安装

安装scrapy

pip安装最方便

查看版本

输出应该是Scrapy 1.2.1，也是官网的最新版本。
