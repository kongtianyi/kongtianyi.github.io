领袖婚礼归来碎碎念

发表于 2021-09-26 | 分类于 闲情偶记 | |

累，不想整理结构，赶紧写完休息了。

简要概括这两天行程，周六4点半起来冲个澡收拾收拾5点半骑共享单车出发去车站，6点半到达车站，7点十来分发车，11点到商丘，骑电动车到商丘南，下午1点多从商丘南回菏泽，两点半到，休息到四点多，吃饭，抄一会儿歌词，7点钟去领袖家。跟领袖和他其他的弟兄们一起吃个饭，到家帮忙布置布置婚房到大概9点多。亲戚都走后跟领袖有一段独处的时间，跟他聊了聊最近关于他婚礼的想法。10点多被安排到旅馆，结束这一天。第二天4点半起来，5点出发去领袖家，6点出发迎亲，迎回来拜堂吃早席，然后在家门口站会儿又出发去办婚礼的地儿，一顿准备，中午婚礼吃饭两点结束。两点出发去机场，3点到等到4点登机，6点到上海，8点到合川路，结束这在路上的两天。

记录几件小事。

先写一件儿好的，在领袖家里一起准备的时候，真是感觉大家都在一个幸福洋溢的氛围里开心的筹备。虽然在我的一贯立场下，很多东西都是繁文缛节，但是在那个氛围里，冰冷的内心确实感到了一种很温馨的感觉。

不随礼而送礼物果不其然受到家里反对，当然这是意料之中。说价值三千的茅台不如随礼一千，而且将来肯定传出去，以后怎么随都不好看。不过我无所谓啦，不想随的我一毛都不会随，礼物代表的是心意，不是价值。如果已经得道成仙了，我更倾向于送法宝仙丹，然而没有，只好送些珍贵的俗物。

领袖的表弟，不是很喜欢，举手投足给人感觉非常的不知礼数。当着我的面，把我刚送给领袖的礼品翻开，当着我的面，跟领袖合计要把我刚送的茅台喝了，我表面平静内心：（敲桌子）喂喂喂喂喂。没大没小的，老摆一出很老成的样子，不喜欢。

不论是早席还是中午的喜宴，菜都太多，根本不可能吃完的程度。我说这是铺张浪费，Z君对我的言论表示匪夷所思，说婚宴这很正常，你结婚时也这样。首先，婚宴上这很正常，我也知道，但这跟铺张浪费根本就是两件事。大家都铺张浪费，所以就不是铺张浪费了吗 = =，所以就合理了吗，所以就不能移风易俗了吗 = = 。另一个，我结婚大抵不会这样，哪怕永远也结不了婚也好。人与人之间的悲欢并不相通，人与人的思想更是千差万别，我只想表述观点，不想争论。

对于新娘，（敲桌子）喂喂喂喂喂，你别……我说加个微信吧，拉你进兄弟群，让兄弟们祝贺祝贺你。不仅不进群，最离谱的是，不给加微信。我擦，你是我请假也要来祝贺的朋友的老婆，按理说今后大家都是朋友吧，我主动加你微信，拒绝？打人不打脸OK？总之我有点生气，因为既然不是朋友，那就是抢夺我们兄弟时间的敌人，彻彻底底的。

感觉自己戾气好重 = = 。
